# Acervo Pessoal

> #### *Projeto final de desenvolvimento de sistemas web, do curso Tecnologia em Análise e Desenvolvimento de Sistemas do IFRN*
>

> #### Deve possuir:
>
> - Cadastro e login
> - Crud de itens
> - Crud de livros
> - Crud de contatos
> - Realizar empréstimos de livro ou item para um contato
> - Registrar devoluções

> #### Tecnologias em uso:
> 
> - HTML
> - CSS
> - Django
> - Python
> - SqlLite
